About This Demo

This is a static demo showing what the OTPify Firefox extension looks like when installed. The actual extension allows you to:

  • Add and manage TOTP and RSA tokens
  • See current and next tokens
  • Track token expiration with a color-changing progress bar
  • Use a dark or light theme that matches your browser

Installation Instructions

To install the actual extension in Firefox:

  1. Download the extension files
  2. Open Firefox and navigate to about:debugging
  3. Click "This Firefox" in the sidebar
  4. Click "Load Temporary Add-on..."
  5. Select any file in the extension directory
